# In-Silico-Docking-Analysis-of-Aspirin-with-SARS-CoV-2-Mpro
Molecular docking study of Aspirin with SARS-CoV-2 Mpro (PDB: 9O6F) using AutoDock Vina and UCSF ChimeraX. Evaluates binding affinity, docking poses, and interaction stability, providing a reproducible pipeline for protein–ligand analysis.
